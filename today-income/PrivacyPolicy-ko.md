# 개인정보처리방침 

**최종 업데이트: 2026년 3월 26일**

**핵심 원칙:** Today's Income은 순수 로컬 앱입니다. 모든 데이터는 사용자의 기기에만 저장됩니다. 당사는 사용자의 개인 데이터를 수집, 전송 또는 공유하지 않습니다.

## 한국어 버전

### 1. 데이터 수집

당사는**사용자 데이터를 수집하지 않습니다**. 이 앱의 모든 데이터는 사용자의 기기에 로컬로 저장됩니다:

- 수입 기록: 사용자의 기기에 있는 로컬 데이터베이스에 저장
- 출퇴근 기록: 사용자의 기기에 있는 로컬 데이터베이스에 저장
- 사용자 설정: 사용자의 기기에 있는 로컬 설정에 저장

당사는 데이터 수집을 위해 분석 도구, 추적 또는 서드파티 SDK를 **사용하지 않습니다**.

### 2. 데이터 저장

당사는 사용자의 모든 데이터(수입 정보, 출퇴근 기록, 근무 시간 등)를 iOS 시스템 표준 저장 메커니즘을 사용하여 사용자의 iOS 기기에 저장합니다. 당사는 사용자의 데이터에 대한 소유권이나 접근권한을 보유하지 않습니다.

### 3. 생체 인증

앱은 앱 내 데이터를 보호하기 위해 Face ID / Touch ID를 지원합니다. 이 기능은 iOS의 네이티브 LocalAuthentication 프레임워크에서 제공합니다. 사용자의 생체 데이터는 당사에서 얻거나 저장되지 않습니다.

### 4. 로컬 알림

앱은 사용자에게 로컬 알림(야근 알림 등)을 보낼 수 있습니다. 이러한 알림은 iOS에서 사용자의 기기에 로컬로 생성됩니다. 당사는 알림 관련 데이터를 수집하지 않습니다.

### 5. 데이터 전송

**이 앱은 네트워크 요청 기능을 포함하지 않습니다.** 당사는 사용자의 데이터를 어떠한 서버에도 전송하지 않습니다. 사용자가 입력한 모든 정보는 기기에만 저장되며 아무 곳으로도 전송되지 않습니다.

### 6. 서드파티 서비스

이 앱은 어떠한 서드파티 서비스, 광고 또는 트래커도 사용하지 않습니다.

### 7. 아동 개인정보

당사는 아동의 정보를 포함한 어떠한 개인정보도 수집하지 않습니다.

### 8. 개인정보처리방침 업데이트
