# Datenschutzrichtlinie

**Zuletzt aktualisiert: 26. März 2026**

**Kernprinzip:** Today's Income ist eine rein lokale App. Alle Daten werden nur auf Ihrem Gerät gespeichert. Wir erheben, übermitteln oder teilen keine Ihrer persönlichen Daten.

## Deutsche Version

### 1. Datenerhebung

Wir **erheben keine** Benutzerdaten. Alle Daten in dieser App werden lokal auf Ihrem Gerät gespeichert:

- Einkommensaufzeichnungen: in der lokalen Datenbank auf Ihrem Gerät gespeichert
- Stempelaufzeichnungen: in der lokalen Datenbank auf Ihrem Gerät gespeichert
- Benutzereinstellungen: in der lokalen Konfiguration auf Ihrem Gerät gespeichert

Wir **verwenden keine** Analysetools, Tracking oder Drittanbieter-SDKs zur Datenerhebung.

### 2. Datenspeicherung

Alle Ihre Daten (Einkommensinformationen, Stempelaufzeichnungen, Arbeitsstunden usw.) werden auf Ihrem iOS-Gerät mit iOS-Standard-Datenspeichermechanismen gespeichert. Wir besitzen oder greifen auf keine Ihrer Daten zu.

### 3. Biometrische Authentifizierung

Die App unterstützt Face ID / Touch ID zum Schutz Ihrer Daten in der App. Diese Funktion wird vom nativen LocalAuthentication-Framework von iOS bereitgestellt. Ihre biometrischen Daten werden von uns niemals erfasst oder gespeichert.

### 4. Lokale Benachrichtigungen

Die App kann Ihnen lokale Benachrichtigungen senden (z.B. Überstundenerinnerungen). Diese Benachrichtigungen werden lokal auf Ihrem Gerät von iOS generiert. Wir erheben keine benachrichtigungsbezogenen Daten.

### 5. Datenübertragung

**Diese App enthält keine Netzwerkanfragefunktionen.** Wir senden keine Ihrer Daten an any Server. Alle Informationen, die Sie eingeben, werden nur auf Ihrem Gerät gespeichert und werden nirgendwohin übertragen.

### 6. Drittanbieterdienste

Diese App verwendet keine Drittanbieterdienste, Werbung oder Tracker.

### 7. Kinderprivatsphäre

Wir erheben keine persönlichen Informationen, einschließlich Informationen von Kindern.

### 8. Aktualisierungen der Datenschutzrichtlinie

Wenn wir diese Datenschutzrichtlinie aktualisieren, werden Änderungen auf dieser Seite veröffentlicht. Die fortgesetzte Nutzung der App stellt die Annahme der aktualisierten Richtlinie dar.
