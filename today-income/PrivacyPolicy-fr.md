# Politique de confidentialité

**Dernière mise à jour : 26 mars 2026**

**Principe fondamental :** Today's Income est une application purement locale. Toutes les données sont stockées uniquement sur votre appareil. Nous ne collectons, ne transmettons ni ne partageons aucune de vos données personnelles.

## Version française

### 1. Collecte de données

Nous **ne collectons pas** de données utilisateur. Toutes les données de cette application sont stockées localement sur votre appareil :

- Enregistrements de revenus : stockés dans la base de données locale de votre appareil
- Enregistrements de pointage : stockés dans la base de données locale de votre appareil
- Paramètres utilisateur : stockés dans la configuration locale de votre appareil

Nous **n'utilisons pas** d'outils d'analyse, de suivi ou de SDK tiers pour collecter des données utilisateur.

### 2. Stockage des données

Toutes vos données (informations de revenus, enregistrements de pointage, heures de travail, etc.) sont stockées sur votre appareil iOS à l'aide des mécanismes de stockage standard du système iOS. Nous ne possédons ni n'accédons à aucune de vos données.

### 3. Authentification biométrique

L'application prend en charge Face ID / Touch ID pour protéger vos données dans l'application. Cette fonctionnalité est fournie par le framework LocalAuthentication natif d'iOS. Vos données biométriques ne sont jamais obtenues ou stockées par nos soins.

### 4. Notifications locales

L'application peut vous envoyer des notifications locales (telles que des rappels d'heures supplémentaires). Ces notifications sont générées localement sur votre appareil par iOS. Nous ne collectons aucune donnée liée aux notifications.

### 5. Transmission de données

**Cette application ne contient aucune fonctionnalité de requête réseau.** Nous n'enverrons aucune de vos données à aucun serveur. Toutes les informations que vous saisissez sont stockées uniquement sur votre appareil et ne seront transmises nulle part.

### 6. Services tiers

Cette application n'utilise aucun service tiers, publicité ou tracker.

### 7. Confidentialité des enfants

Nous ne collectons aucune information personnelle, y compris les informations provenant d'enfants.

### 8. Mises à jour de la politique de confidentialité

Si nous mettons à jour cette politique de confidentialité, les modifications seront publiées sur cette page. La poursuite de l'utilisation de l'application constitue l'acceptation de la politique mise à jour.
