# 隱私政策 

**最後更新日期：2026年3月26日**

**核心原則：** 今日收入是一款純本地應用。所有數據僅存儲在您的設備上。我們不會收集、傳輸、共享您的任何個人數據。

## 繁體中文版

### 1. 數據收集

我們**不收集**任何用戶數據。本應用的所有數據都存儲在您的設備本地：

- 收入記錄：存儲在您設備上的本地數據庫
- 打卡記錄：存儲在您設備上的本地數據庫
- 用戶設置：存儲在您設備的本地配置中

我們**不使用**任何分析工具、埋點或第三方 SDK 來收集用戶數據。

### 2. 數據存儲

您的所有數據（收入信息、考勤記錄、工作時長等）都存儲在您的 iOS 設備上，採用 iOS 系統標準的數據存儲機制。我們不擁有、不訪問您的任何數據。

### 3. 生物識別

應用支持 Face ID / Touch ID 功能，用於保護您應用內的數據安全。此功能由 iOS 系統原生的 LocalAuthentication 框架提供，您的生物識別數據不會被我們獲取或存儲。

### 4. 本地通知

應用可能向您發送本地通知（如加班提醒）。這些通知由 iOS 系統在您的設備上本地生成，我們不收集任何通知相關的數據。

### 5. 數據傳輸

**本應用不包含任何網絡請求功能。** 我們不會將您的任何數據發送到任何服務器。您輸入的所有信息都僅存儲在您的設備本地，不會被傳輸到任何地方。

### 6. 第三方服務

本應用不使用任何第三方服務、廣告或追蹤器。

### 7. 兒童隱私

我們不會收集任何個人信息，包括兒童的信息。

### 8. 隱私政策更新

如果我們更新隱私政策，將在此頁面發布。繼續使用應用即表示您接受更新後的政策。
